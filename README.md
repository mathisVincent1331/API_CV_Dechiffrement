# API_CV_Dechiffrement
Cette application fonctionnant côté client permet de décrypter un fichier crypté grâce à l'algorithme AES-CBC.

Pour l'utiliser, il vous suffit tout simplement d'insérer le fichier chiffré envoyé en pièce jointe du mail et le déposer dans la zone prévue à cet effet sur l'application Web. 
Il ne vous restera plus qu'à télécharger le CV déchiffré en appyant sur le bouton en dessous du dépôt, et le tour sera joué!
Si vous ne recevez aucun fichier à cause du pare-feu ou autre sécurité (ce qui est bien en soit :)), voici un lien Drive pour le récuperer facilement.
https://drive.google.com/drive/folders/1NJsSpbntUUZCZA-9HqQVkD05ObV9CNrE?usp=sharing

Avec la concurrence de plus en plus féroce, je me suis demandé comment faire pour proposer une demande originale et audacieuse qui me permettrait de faire valloir mes compétences. Avec cette petite API ludique, j'espère susciter votre intêret envers mon profil d'un étudiant cherchant une alternance en Cybersécurité.

Cette application ne conserve aucune données du côté Client. Elle est safe à utiliser.
